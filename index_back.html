Abstract

Reliable humanoid-object interaction remains difficult because carrying requires balanced locomotion, closed-loop perception, and integration with high-level planners. Pro-HOI introduces a root-guided RL framework that conditions control on desired root trajectories while using motion tracking only as reward, making the root a universal interface for navigation and obstacle avoidance. A persistent object estimation module fuses real-time detection with a digital twin to detect drops and trigger autonomous re-grasping, enabling long-horizon robustness. Deployed on a Unitree G1 with onboard sensing and computing, Pro-HOI outperforms baselines in generalization and achieves stable multi-cycle box carrying in complex scenes.
